Automation Create Group and Post Using Selenium
===============================================

Introduction
------------

This is a Python script that uses Selenium to automate the process of creating a Facebook group and posting content to the group.

Installation
------------

1.  Install Python
2.  Install the following Python libraries: pip  install pandas selenium webdriver-manager
3.  Download and install Google Chrome browser
4.  Download the ChromeDriver executable that matches your Chrome browser version and operating system, and save it to your local machine

Usage
-----

1.  Open the script in your cmd/terminal
2.  Update the Facebook credentials, group details, and post content in the script
3.  Run the script

Status content
--------------

For status content, if you want to make new line or enter, just add \n in your every pharagraph.
